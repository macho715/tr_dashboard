# Dashboard Layout Renewal

## Purpose
- Reorder information architecture with a sticky summary bar, left navigation, and a three-stage main flow.
- Separate Weather/Notice/KPI/Logs for faster operational visibility.

## Structure
- **Sticky summary bar**: core KPIs + Notice/Weather update state + risk badge.
- **Left navigation**: Timeline/Voyages/Weather/KPI/Notice/Logs sections.
- **Three-stage main**: Timeline (Gantt) → Voyage + Schedule → Weather + Heatmap.
- **Operations aids**: Data Consistency badge and Pipeline Status (A~N).

## View modes
- **standard**: left/right sidebars on.
- **compact**: right sidebar hidden.
- **fullscreen**: both sidebars hidden for briefing focus.
