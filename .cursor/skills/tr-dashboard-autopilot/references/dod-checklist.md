# DoD Checklist (AGENTS.md 기반)

- Story Header: TR 선택 후 3초 내 Where/When&What/Evidence 요약
- Root cause: 2클릭으로 Why→Evidence/History 도달
- Reflow: Preview→Apply 분리 + Apply 시 plan_reflow_applied 기록
- 모드: Live/History/Approval/Compare 분리(권한 포함)
- Collision: 분류/Severity/해결 옵션/근거 제공
- SSOT 준수: Activity 변경은 option_c.json만
- 감사성: History append-only, Evidence 미비 시 전이 차단
